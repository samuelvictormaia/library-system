## üöÄ Tech Stack

- **Language:** TypeScript  
- **Runtime:** Node.js  
- **Paradigm:** Object-Oriented Programming (OOP)  
- **Key Concepts:** Classes, Async/Await, Promises, Error Handling, Map Collections  

---

## üß© Features

- Add books to a digital library collection  
- Borrow books asynchronously (simulating database delay)  
- Return borrowed books with validation  
- Prevent duplicate entries and invalid returns  
- Display all books with availability status  

---

## üß† Core Concepts Demonstrated

| Concept | Description |
|----------|--------------|
| **Encapsulation** | Each book is represented as a `Book` class with private and controlled properties. |
| **Abstraction** | The `BookManager` class manages all library logic through clean, reusable methods. |
| **Async Simulation** | Borrowing and returning use Promises to simulate async data operations. |
| **Error Handling** | Robust validation and clear feedback for invalid or duplicate actions. |
